# Charla 🦜 — Learn Spanish

A free, original, gamified Spanish-learning web app (PWA). Built as a personal learning tool.

**Live app:** https://saltybayflips-ai-empire.github.io/charla/

## Install on iPhone
1. Open the link above in **Safari**
2. Tap **Share** → **Add to Home Screen**
3. Launch Charla from your home screen — it runs fullscreen and works offline

## What's inside
- 15-unit Spanish course (3 sections, ~180 words, ~150 sentences, 5 illustrated stories)
- 8 exercise types: translate (word-bank or typing), listening, type-what-you-hear, fill-in-the-blank, multiple choice, picture match, pair matching, speaking (with mic)
- Spanish audio via on-device text-to-speech (normal + slow turtle speed)
- Hearts, gems, streaks + streak freezes, daily quests, monthly challenge, XP boosts
- Weekly leagues (Bronze → Diamond) with promotion/demotion
- Achievements, mistakes review, smart practice, words list with strength tracking
- Progress saved on-device (localStorage) with backup/restore codes

## Tech
Vanilla HTML/CSS/JS, no build step, no backend. Service worker for offline. All content and artwork original.

*Personal project — not affiliated with, endorsed by, or connected to Duolingo or any other language-learning service.*
